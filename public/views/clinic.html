PUPPY VACCINATIONS
Puppy vaccination sets must be done 3 to 4 weeks apart. If you go longer than 4 weeks between any sets, it is recommended to repeat the last set. If your puppy received vaccinations at younger than 8 weeks, it is considered an extra set and it will be necessary to give a total of 4 sets instead of 3.
First SET given at 8-12 Weeks: $27.50 First Distemper/Parvo Combination (includes Adenovirus Type 2 and Parainfluenza) First Deworm for intestinal parasites (Pyrantel for round worms)
Second SET given at 12-16 Weeks: $35.50 Second Distemper/Parvo Combination First Bordetella (kennel cough) given intranasally Second Deworm
Third SET given at 16-20 Weeks: $35.00 Annual Distemper/Parvo Combination Annual Bordetella One year Rabies (must be at least 12 weeks old to receive)
